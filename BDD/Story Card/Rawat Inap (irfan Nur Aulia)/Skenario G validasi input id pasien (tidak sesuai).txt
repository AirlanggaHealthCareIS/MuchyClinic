Fitur : Admin memasukkan data pasien untuk melakukan rawat inap
Skenario F	: Ketika menekan tombol search, ternyata input ID_pasien tidak sesuai, bisa karena ID_pasien yang di inputkan salah atau ID_pasien tidak ada di tabel pemeriksaan / pasien
	Given	: Pasien diharuskan melakukan rawat inap.
	And	: Admin telah melakukan login.
	And 	: Admin berada home page “Muchy Clinic”.
	When	: Admin memilih menu rawat inap.
	Then	: Admin berada halaman rawat inap.
	When	: Admin memasukkan ID pasien.
	Then	: Admin menerima pemberitahuan bahwa inputan ID_pasien belum terdaftar. (danger/warning).
	And	: Admin mengisi kembali Id Pasien.
